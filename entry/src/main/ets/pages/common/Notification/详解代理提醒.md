# 代理提醒

我来帮你查询HarmonyOS中实现定时通知的相关知识和最佳实践。

现"到点发通知"功能，核心是使用**代理提醒**能力。以下是完整的实现思路：

---

## 一、核心概念

**代理提醒**是HarmonyOS提供的后台代理提醒能力，当应用被冻结或退出后，计时和提醒功能由系统服务代理执行，确保定时通知可靠触发。

**支持三种提醒类型：**
| 类型       | 说明               | 适用场景               |
| ---------- | ------------------ | ---------------------- |
| **倒计时** | 基于秒数的提醒     | 临时提醒、会议倒计时   |
| **日历**   | 基于日期时间的提醒 | 生日、纪念日、日程     |
| **闹钟**   | 基于时钟的提醒     | 每日起床闹钟、打卡提醒 |

---

## 二、实现步骤

### 步骤1：申请权限

在 `module.json5` 中声明权限：

```json
{
  "module": {
    "requestPermissions": [
      {
        "name": "ohos.permission.PUBLISH_AGENT_REMINDER",
        "reason": "$string:reminder_reason",
        "usedScene": {
          "abilities": ["EntryAbility"],
          "when": "always"
        }
      }
    ]
  }
}
```

### 步骤2：请求通知授权

```typescript
import { notificationManager } from '@kit.NotificationKit';
import { BusinessError } from '@kit.BasicServicesKit';

// 请求通知授权
notificationManager.requestEnableNotification().then(() => {
  console.info('通知授权成功');
}).catch((err: BusinessError) => {
  console.error(`通知授权失败: ${err.code}, ${err.message}`);
});
```

### 步骤3：定义并发布提醒

```typescript
import { reminderAgentManager } from '@kit.BackgroundTasksKit';
import { notificationManager } from '@kit.NotificationKit';
import { BusinessError } from '@kit.BasicServicesKit';

// 方式一：闹钟提醒（每天固定时间）
async function setAlarmReminder() {
  let alarm: reminderAgentManager.ReminderRequestAlarm = {
    reminderType: reminderAgentManager.ReminderType.REMINDER_TYPE_ALARM,
    hour: 8,          // 8点
    minute: 0,        // 0分
    daysOfWeek: [1, 2, 3, 4, 5],  // 周一到周五
    title: '起床闹钟',
    content: '该起床了！',
    actionButton: [
      {
        title: '关闭',
        type: reminderAgentManager.ActionButtonType.ACTION_BUTTON_TYPE_CLOSE
      },
      {
        title: '稍后提醒',
        type: reminderAgentManager.ActionButtonType.ACTION_BUTTON_TYPE_SNOOZE
      }
    ],
    wantAgent: {
      pkgName: 'com.example.myapp',
      abilityName: 'EntryAbility'
    },
    ringDuration: 30,      // 响铃时长（秒）
    snoozeTimes: 3,        // 延迟提醒次数
    timeInterval: 300,     // 延迟提醒间隔（秒）
    notificationId: 100,
    slotType: notificationManager.SlotType.CONTENT_INFORMATION
  };

  try {
    let reminderId = await reminderAgentManager.publishReminder(alarm);
    console.info(`闹钟发布成功，ID: ${reminderId}`);
  } catch (err) {
    console.error(`闹钟发布失败: ${(err as BusinessError).message}`);
  }
}

// 方式二：日历提醒（指定日期时间）
async function setCalendarReminder() {
  let calendar: reminderAgentManager.ReminderRequestCalendar = {
    reminderType: reminderAgentManager.ReminderType.REMINDER_TYPE_CALENDAR,
    dateTime: {
      year: 2024,
      month: 12,
      day: 25,
      hour: 9,
      minute: 0
    },
    title: '会议提醒',
    content: '您有一场会议即将开始',
    actionButton: [
      {
        title: '关闭',
        type: reminderAgentManager.ActionButtonType.ACTION_BUTTON_TYPE_CLOSE
      }
    ],
    wantAgent: {
      pkgName: 'com.example.myapp',
      abilityName: 'EntryAbility'
    },
    notificationId: 101,
    slotType: notificationManager.SlotType.CONTENT_INFORMATION
  };

  let reminderId = await reminderAgentManager.publishReminder(calendar);
  console.info(`日历提醒发布成功，ID: ${reminderId}`);
}

// 方式三：倒计时提醒（指定秒数后触发）
async function setTimerReminder() {
  let timer: reminderAgentManager.ReminderRequestTimer = {
    reminderType: reminderAgentManager.ReminderType.REMINDER_TYPE_TIMER,
    triggerTimeInSeconds: 300,  // 5分钟后提醒
    title: '倒计时提醒',
    content: '时间到了！',
    wantAgent: {
      pkgName: 'com.example.myapp',
      abilityName: 'EntryAbility'
    },
    notificationId: 102
  };

  let reminderId = await reminderAgentManager.publishReminder(timer);
  console.info(`倒计时发布成功，ID: ${reminderId}`);
}
```

### 步骤4：取消提醒

```typescript
// 取消指定提醒
async function cancelReminder(reminderId: number) {
  await reminderAgentManager.cancelReminder(reminderId);
  console.info('提醒已取消');
}

// 取消所有提醒
async function cancelAllReminders() {
  await reminderAgentManager.cancelAllReminders();
  console.info('所有提醒已取消');
}

// 获取所有有效提醒
async function getValidReminders() {
  let reminders = await reminderAgentManager.getValidReminders();
  console.info(`当前有效提醒数量: ${reminders.length}`);
}
```

## 三、关键配置说明

| 配置项           | 说明                          |
| ---------------- | ----------------------------- |
| `wantAgent`      | 点击通知后跳转的目标Ability   |
| `actionButton`   | 通知上的按钮（关闭/稍后提醒） |
| `ringDuration`   | 响铃时长（秒）                |
| `snoozeTimes`    | 延迟提醒次数                  |
| `timeInterval`   | 延迟提醒间隔（秒）            |
| `notificationId` | 通知ID，相同ID会覆盖          |
| `slotType`       | 通知渠道类型                  |

---

## 四、注意事项

1. **权限申请**：手机/平板设备需要向华为申请代理提醒权限（通过AppGallery Connect申请）
2. **数量限制**：单个应用有效提醒不超过30个
3. **场景限制**：仅支持生活类、运动健康类、学习类、办公类等场景，不支持营销类场景
4. **跳转限制**：点击通知只能跳转到申请代理提醒的本应用

---

## 五、完整流程图

```
用户设置提醒时间
       ↓
请求通知授权
       ↓
定义ReminderRequest（闹钟/日历/倒计时）
       ↓
调用publishReminder发布提醒
       ↓
系统代理计时（应用可退出）
       ↓
到达设定时间 → 系统发布通知 → 用户收到提醒
```

这种方案的优势是**即使应用退出或被冻结，提醒也能准时触发**，由系统服务代理执行，无需应用保活。
